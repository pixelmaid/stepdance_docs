\doxysubsection{Detailed Description}
\doxylink{class_output_port}{Output\+Port}. 

Output\+Ports are modules that convert internal step commands into a frame of pulse signals on the physical output port on the {\bfseries{machine controller module}} or {\bfseries{basic module}}. Output ports contain step and direction signal flags for each of the six output signal types (X, Y, Z, E, R, and T).
