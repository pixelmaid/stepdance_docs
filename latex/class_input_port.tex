\doxysubsection{Detailed Description}
\doxylink{class_input_port}{Input\+Port}. 

\doxylink{class_input_port}{Input\+Port} components receive motion streams on physical Stepdance input ports, and map these signals to downstream components. Receiving and decoding of input streams occurs asynchronously.
